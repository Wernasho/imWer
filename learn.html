¿Qué son las Arrow Functions en JavaScript?

Las Arrow Functions son una forma más corta de escribir funciones en JavaScript

Comparación entre una función "nomral" y una Arrow Function:


//Función normal
function displayMessage(name, message) {
    console.log($`{name}`: ${message});
}
//Arrow Function:

const displayMessage = (name, message) => `${name}: ${message}`;
console.log(displayMessage("wer", "esto es una arrow function"));

¿Qué significa todo eso?

A ver, vamos a ir por partes.

const displayMessage = (name, message)

En esa línea declaramos la variable displayMessage y le asignamos una arrow function.

Luego, los parentesis, tal como en una función nomral, contienen los parámetros de la función.

 => `${name}: ${message}`;

Lo que hace esta línea es devolver el valor que esté luego de =>, ya que al ser solo un valor, JavaScript lo devuelve automáticamente sin necesidad de usar return
